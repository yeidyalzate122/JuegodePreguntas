Bienvenidos
El proyecto funciona totalmente en consola.
El juego está dividido por las siguientes categorías:
•	primera: acertijos de animales
•	segunda: problemas de razonamiento lógico
•	tercera: problemas básicos se ciencias naturales y química
•	cuarto: acertijos de naturaleza
•	quinto: acertijos de instrumentos musicales
Disfruten mucho este juego. 

Para iniciar el juego debes de ir a la carpeta principal llamada "Preguntas y respuestas", y ahi esta el metodo principal para iniciarlo
